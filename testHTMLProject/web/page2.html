목록 만들기 연습

ul(unorder list), ol(order list), dl(definition list)

ul : 순번없는 목록, 항목 앞에 블릿기호 표시

  • JAVA
  • ORACLE
  • JDBC
  • HTML5

ol : 순번이 매겨지는 목록

  1. HTML5
  2. CSS3
  3. JAVASCRIPT
  4. JQUERY
  5. WEB API

dl : 제목과 설명을 하나로 묶어서 목록을 만듦

html5
w3c에서 발표한 웹 표준 2.0의 기술을 제공하는 새로운 웹페이지 제작용 마크업 언어이다.
css3
html5와 함께 웹 표준 2.0의 웹페이지의 스타일을 적용하기위한 스타일시트 새 버전이다.
javascript
자바로 만든 웹 페이지에 사용되는 스크립트 언어이다.
jquery
jquery 사에서 자바스키립트 소스를 작성해서 제공한 오픈소스들을 말한다.
